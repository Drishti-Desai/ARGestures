ARGestures is a fun computer vision project built using Python, OpenCV, and MediaPipe.

This project tracks hand gestures in real time using a webcam and transforms them into interactive AR-style effects such as:

✍️ Air Drawing
🕷️ Spider-Man Web Shooting Effects
✋ Gesture-Based Screen Clearing

The project was created as a fun side project to explore:
Hand tracking
Gesture recognition
Computer vision
Real-time visual effects

Features :
Real-time hand tracking
Air drawing using finger gestures
Spider-Man inspired web animation effects
Gesture-controlled canvas clearing
Smooth AR-style interactions
Multi-hand detection support


Concepts Used : 
Computer Vision
Hand Landmark Detection
Gesture Recognition
Real-Time Rendering
Augmented Reality Effects


Built as a fun side project to experiment with AR effects, hand tracking, and real-time gesture recognition.
